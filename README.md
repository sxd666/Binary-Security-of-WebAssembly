# Binary Security of WebAssembly

组号：G2

组员：梁郅卓，李要杰，沈新迪

## 项目选题

**WebAssembly**的二进制安全性

## 项目简介

**WebAssembly**或称**wasm**是一个低级编程语言，旨在通过严格分离代码和数据、强制类型和限制间接控制流，在浏览器和其他平台上安全可靠地运行代码。

但是，不安全原语中的漏洞仍然可以转化为**wasm**二进制文件中的漏洞。

因此，本项目旨在调研**wasm**二进制漏洞的相关工作，

1. **wasm**的特性
2. **wasm**在内存上的结构和机制
3. 部分二进制漏洞的复现
